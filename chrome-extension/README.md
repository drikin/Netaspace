# Backspace.fm ネタ投稿 Chrome Extension

現在閲覧中のページを簡単にbackspace.fmのネタとして投稿できるChrome拡張機能です。

## 機能

- **ワンクリック投稿**: ツールバーのアイコンをクリックするだけでポップアップが開きます
- **自動情報取得**: ページタイトル、URL、説明文を自動で取得します
- **設定保存**: 投稿者名とサーバーURLを保存して次回以降の入力を省略
- **フィンガープリント生成**: ユーザー識別用のフィンガープリントを自動生成

## インストール手順

### 1. 拡張機能ファイルの準備

1. `chrome-extension` フォルダ全体をダウンロードします
2. アイコンファイルを作成します：
   - `chrome-extension/create-icons.html` をブラウザで開く
   - 表示された3つのキャンバスを右クリックして画像として保存
   - `icons/icon16.png`, `icons/icon48.png`, `icons/icon128.png` として保存

### 2. Chrome拡張機能として読み込み

1. Chrome ブラウザを開く
2. アドレスバーに `chrome://extensions/` を入力
3. 右上の「デベロッパーモード」をオンにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. `chrome-extension` フォルダを選択

### 3. 初期設定

1. 拡張機能が読み込まれたら、ツールバーのアイコンをクリック
2. サーバーURL欄にあなたのbackspace.fmアプリのURLを入力
   - 例: `https://your-app-name.replit.dev`
3. 投稿者名を入力（これらの設定は保存されます）

## 使用方法

1. ネタにしたいページを開く
2. ツールバーのBackspace.fmアイコンをクリック
3. 必要に応じてタイトルや説明を編集
4. 「ネタを投稿」ボタンをクリック

## ファイル構成

```
chrome-extension/
├── manifest.json          # 拡張機能の設定ファイル
├── popup.html             # ポップアップのHTML
├── popup.js              # ポップアップの動作制御
├── content.js            # ページ情報取得スクリプト
├── create-icons.html     # アイコン生成用ヘルパー
├── icons/
│   ├── icon16.png       # 16x16 アイコン
│   ├── icon48.png       # 48x48 アイコン
│   └── icon128.png      # 128x128 アイコン
└── README.md            # このファイル
```

## セキュリティ

- 拡張機能は現在のタブの情報のみにアクセスします
- 投稿者名とサーバーURLのみがChromeの同期ストレージに保存されます
- フィンガープリントはブラウザの特性から生成される一意の識別子です

## トラブルシューティング

### 投稿が失敗する場合

1. サーバーURLが正しく設定されているか確認
2. サーバーが稼働しているか確認
3. CORSエラーが発生する場合は、サーバー側でCORS設定を確認

### ページ情報が取得できない場合

一部のサイトではセキュリティ制限により情報取得ができない場合があります。その場合は手動でタイトルと説明を入力してください。

## 開発者向け情報

### 権限

- `activeTab`: 現在のタブの情報取得
- `storage`: 設定の保存
- `host_permissions`: 指定したサーバーへのリクエスト送信

### API エンドポイント

拡張機能は以下のエンドポイントを使用します：
- `POST /api/topics` - ネタの投稿

### カスタマイズ

`manifest.json` の `host_permissions` を編集することで、接続可能なサーバーを制限できます。
